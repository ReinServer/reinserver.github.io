connection reset

this is an issue that is a pain to deal with. if you're here, we're sorry.


• Ensure that the IP and port is correct. For java, use play.reinserver.com. For bedrock, use play.reinserver.com as the IP and 19132 as the port.

• Ensure that you have a steady internet connection that isn't dropping

• Try joining other servers that are not running on a proxy. To check this, press F3 and check if the server is not running on BungeeCord, Velocity or Waterfall.


if you can join other servers that are not running on a proxy:

One fix that has worked for many is to install Fabric Loader and the Simple Voice Chat mod.

Note that this fix will only work on servers that support Simple Voice Chat.

For a tutorial on installing Simple Voice Chat, go to reinserver.com/vc


If that didn't work, then:

• Update Java

• Decrease your render distance, then join

• Use a different WiFi network or use Mobile Data (charges may apply)

• Renew your IP

• Change your DNS server

• Use a VPN


If the above still didn't work, it could be your Internet Server Provider limiting the speed of your connections. Contact your ISP for more info.


If you cannot join other servers that are not running on a proxy:

• Restart Minecraft

• Restart your computer/device

• Restart your router

• Disable your firewall

• Use a vanilla client (if you're running a modified client)


If the above still didn't work, join our Discord or WhatsApp for more help, or check "can't connect to server"
